### 💳 [Neural Fraud Shield](https://github.com/prathameshsinarkar27/Neural-Fraud-Shield)
Financial fraud detection system combining deep learning and gradient boosting, with full
prediction-level explainability.

`Python` `TensorFlow` `XGBoost` `SHAP` `LIME` `Flask`

- Deep Neural Network + XGBoost ensemble on 284K+ real-world transactions
- Achieved ROC-AUC of 0.98 despite 0.17% class imbalance (SMOTE-balanced)
- Live-scoring Flask API with SHAP/LIME explainability and a real-time dashboard
- 🔗 [Live Demo](https://neural-fraud-shield.onrender.com/)
